03.04.01 Design Sprint

About: A group assignment in which we created an app for international students that are looking for jobs, we called it MYK (Monetize Your Knowledge).
How: Create an Android compatible app starting from nothing, using the Design Sprint method. We had to define a long term goal, create a high-fidelity interactive prototype that acts as proof-of-concept, using UI kits. We pitched it in front of the class.
Learning objectives: Teamwork, developing using Design Sprint method, creating a product from just an idea.
My role: "Driver", participated with ideas, made sure the team kept the right direction and same quality of work, implemented part of the prototype.
